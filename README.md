# trump-analyst

A Claude skill for analyzing and predicting Donald Trump's behavior through the lens of negotiation psychology and behavioral pattern recognition.

## What It Does

This skill turns Claude into a behavioral analyst for Trump's political moves. Instead of just describing what Trump did, it connects actions to recurring negotiation patterns and pushes toward **predictive insight** — what comes next, and why.

### Coverage Areas

- **Predictive Modeling** — "What will Trump do about X?" answered through a structured behavioral framework
- **Negotiation Psychology** — Anchoring, walkaway leverage, pain threshold testing, counter-party reading, deal pivots
- **Rhetoric & Persuasion** — Nickname strategy, rally cadence, media judo, emotional sequencing
- **Policy Analysis** — Tariffs, immigration, foreign policy framed as negotiation moves
- **Social Media Strategy** — Truth Social patterns, ALL CAPS mechanics, narrative threading

### Key Features

- **Neutral lens** — analyzes the *how* and *why* without taking sides
- **Adaptive tone** — matches casual or formal energy depending on your question
- **Web search integration** — pulls current data for recent events, uses existing knowledge for historical patterns
- **Calibrated predictions** — Most likely / If pressured / Wildcard structure with confidence levels

## Example Prompts

```
"What's the logic behind Trump's tariff strategy?"
"How will Trump respond to the EU's retaliatory tariffs?"
"Why does Trump give people nicknames? What's the actual strategy?"
"lol why does he type in ALL CAPS on Truth Social"
"What's Trump's play with China right now?"
```

## Installation

1. Download `trump-analyst.skill`
2. Open Claude → **Settings** → **Skills**
3. Click **Add Skill** and upload the file

The skill triggers automatically when you ask about Trump's rhetoric, policies, social media, or predictive questions.

## Files

| File | Description |
|------|-------------|
| `SKILL.md` | The skill source — all instructions and frameworks |
| `trump-analyst.skill` | Packaged skill file ready to install |

## Modifying

To customize the skill:

1. Edit `SKILL.md` to your liking
2. Ask Claude (with the skill-creator skill) to repackage it: *"repackage my trump-analyst skill"*
3. Or manually zip the folder and rename to `.skill`
